## How to Install a Wallet

### For Browser Extensions (Freighter, xBull, Albedo, etc.)

1. **Click "Install"** button in the wallet selector modal
2. **You'll be redirected** to the wallet's official website or extension store
3. **Click "Add to Browser"** or "Install"
4. **Confirm the installation** when prompted
5. **Create a new wallet** or import existing one
6. **Save your secret key/seed phrase** securely (never share it!)
7. **Return to TrustLance** and click "Connect"

### For Mobile Wallets (LOBSTR, Hana, Klever)

1. **Open App Store** (iOS) or **Play Store** (Android)
2. **Search for the wallet name**
3. **Download and install** the app
4. **Create a new wallet** or import existing one
5. **Save your secret key/seed phrase** securely
6. **Use browser connection** if supported, or use mobile browser

## First Time Setup

### Step 1: Install Freighter (Recommended)

```
1. Visit: https://www.freighter.app/
2. Click "Add to Chrome" (or your browser)
3. Click "Add Extension"
4. Create new account or import existing
5. Save your secret key securely!
```

### Step 2: Fund Your Wallet

```
1. Open Freighter extension
2. Click "Receive" to get your address
3. Copy your Stellar address (starts with G...)
4. Transfer XLM from exchange or friendbot
```

**Get Testnet XLM:**
- Visit: https://laboratory.stellar.org/#account-creator
- Enter your address
- Click "Create Account"
- You'll receive 10,000 test XLM

### Step 3: Connect to TrustLance

```
1. Click "Connect Wallet" on TrustLance
2. You'll see Freighter with "Installed" badge
3. Click "Connect"
4. Approve the connection in Freighter popup
5. Done! Your wallet is connected
```

## Troubleshooting

### "Wallet not installed" error

**Solution:**
1. Make sure you installed the wallet extension
2. Refresh the page after installation
3. Check if extension is enabled in your browser
4. Try clicking "Install" button again

### Connection popup doesn't appear

**Solution:**
1. Check if popup blocker is enabled
2. Allow popups for TrustLance
3. Try clicking Connect again
4. Make sure wallet extension is unlocked

### Wrong network error

**Solution:**
1. Open your wallet settings
2. Switch to "Testnet" network
3. Reconnect to TrustLance

### Can't see installed wallets

**Solution:**
1. Refresh the page
2. Clear browser cache
3. Make sure wallet extension is enabled
4. Try a different browser

## Security Tips

⚠️ **Never share your secret key or seed phrase!**

✅ **Best Practices:**
- Store secret keys offline
- Use hardware wallets for large amounts
- Enable all security features
- Keep wallet software updated
- Use strong passwords
- Backup your keys securely

❌ **Avoid:**
- Sharing screenshots with keys
- Entering keys on websites
- Storing keys in cloud/email
- Using public WiFi for transactions
